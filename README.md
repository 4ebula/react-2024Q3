## Class components

- To lauch project use
```bash
npm run dev
```

Application will be served on port [5173](http://localhost:5173/)

- To lint code, use
```bash
npm run lint
```

Also, code is automatically lined on commits

- To foramt code, use
```bash
npm run format:fix
```

## Tests
- To execute tests, run
```bash
npm run test
```

- To display coverage, run
```bash
npm run test:coverage
```
